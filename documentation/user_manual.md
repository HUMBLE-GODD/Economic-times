# User Manual

## Workflow

1. Sign in.
2. Review Mission Control KPIs and alerts.
3. Open Risk Intelligence to inspect compound risk.
4. Use Operations to investigate incident patterns.
5. Use Permits to review a work permit against controls and RAG evidence.
6. Use Maintenance and Assets to prioritize equipment work.
7. Use Compliance to inspect OSHA ITA-derived review records.
8. Use Digital Twin to visualize zones, assets, sensors, and simulated propagation.
9. Use AI Copilot to run a specialized agent.
10. Generate executive, risk, maintenance, incident, or compliance reports.

The demo dataset is derived from Phase 1 public datasets. It is not a customer plant data dump.
